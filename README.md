# Robot-Learning-for-Touch-Screen
It is a robot learning project to record user data, train the data, simulate the data and evaluate user trust on robots with human data trusted robot.

## GUI for data collection

In the corresponding ws run

ros2 run rl_gui rl_gui

it will publish a topic with interested positions

## Arduino for force collection

Use Arduino IDE for regular force data reading and use serial reader to be able to read arduino terminal data in ros2
